# Project Overview

This repository is a personal educational workspace for learning and documenting JavaScript, TypeScript, CSS, and data structures / algorithms.

## What this project contains

- `cascading-style-sheets/`
  - Notes and examples for CSS selectors, pseudo-classes, pseudo-elements, and CSS functions.

- `javascript-sheet/`
  - A collection of JavaScript topic files covering core language fundamentals, utilities, modern features, DOM and event handling, data types, functions, arrays, objects, and common algorithm patterns.

- `js-for-algorithms/`
  - Structured algorithm and data structure content.
  - Includes explanations for searching, sorting, recursion, dynamic programming, and interview-style algorithm problems.
  - Contains implementations of common LeetCode problems and core data structure examples.

- `typescript-sheet/`
  - A sequential TypeScript learning path from basics through advanced topics.
  - Includes files for types, type inference, interfaces, generics, decorators, classes, modules, error handling, testing, React integration, Node.js/Express, and more.
  - Also includes configuration files like `tsconfig.json`, `jest.config.js`, and a TypeScript cheat sheet.

## Purpose of the work

The goal is to build a reusable learning repository that:

- Documents concepts with clear, hands-on examples.
- Helps practice implementation of algorithms and data structures.
- Supports transitioning from JavaScript fundamentals to advanced TypeScript development.
- Provides a reference for building real apps with modern language features.

## How to use it

1. Browse the folder structure to find the topic you want to study.
2. Open the matching `.js`, `.ts`, or `.md` file to read examples and explanations.
3. Run TypeScript samples from `typescript-sheet/` using the TypeScript compiler and optionally `jest` for tests.

## Notes

- This repository is organized as a personal study resource.
- Files are grouped by topic and learning progression rather than production app structure.
- The current focus is on strengthening fundamentals and exploring algorithmic thinking.
